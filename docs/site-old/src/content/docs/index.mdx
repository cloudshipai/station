## Quick Start

```bash
# Install Station
curl -sSL https://getstation.cloudshipai.com | bash

# Initialize and create your first agent
stn init
stn agent create --name "System Monitor" --description "Monitor system resources"

# Test your agent
stn agent run 1 "Check current disk usage and memory status"
```
